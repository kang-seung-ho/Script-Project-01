이름: 강승호

 

1) 프로젝트 이름
File Assistant


 

2) 프로젝트 개발 동기
휴대폰 사진을 분류하고 싶은데 너무 많을때, 바탕화면에 너무 파일이 많아서 정리하고 싶을때, 파일 내용은 아는데 무슨 파일인지 모를때 이런 프로그램이 있으면 유용할 것 같아서 개발하게 되었습니다.
 

 

3) 프로그램 주요 기능
1. 폴더 선택 후 사진 파일의 데이터를 읽어와 그 사진에 대한 연/월에 해당되는 폴더가 없다면 생성한 뒤 사진을 그 폴더로 옮깁니다.
2. 폴더 선택 후 사용자가 설정한 기간을 넘으면 OLD 폴더로 옮깁니다. OLD폴더를 압축하는 기능을 추가해 안쓰는 파일들을 삭제하고 압축파일로 만듭니다.
3. 폴더를 선택하고 텍스트 입력 후 검색버튼을 누르면 해당 폴더의 파일들의 내용을 탐색합니다.RESULT폴더를 생성해 파일을 복사합니다. 예시) 소스.py 내의 내용까지 검색 하여 입력한 단어가 포함되어 있다면 RESULT 폴더로 복사
4. 폴더 선택 후 확장자 입력 시 입력한 확장자에 해당하는 파일들을 확장자명으로 생성된 폴더에 복사합니다.
5. 문서 파일 선택 시 해당 파일을 PDF로 변환해주는 기능을 통해 보다 편리하게 필기/제출용 PDF 파일을 만들 수 있습니다.


4) 개발 방법
Pycharm 개발환경에서 CustomTkinter를 활용해 File Reading과 Organizing Files, Tikinter에서 배운 내용들로 개발합니다. 필요한 추가적인 라이브러리를 설치해 파일 관리 프로그램에 활용합니다.
